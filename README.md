# DropMoney2 - A Economy plugin for Spigot Minecraft Server!

You can get money when you destory mobs.

## Instllation
[DropMoney2.jar](https://github.com/Fnit-Crafters/DropMoney2/raw/master/out/artifacts/DropMoney2_jar/DropMoney2.jar) to your `plugins` directory.

## Dependence
Vault (https://dev.bukkit.org/projects/vault)
